# Not Found Page

The `not-found.js` file lets you define a custom interface for non-existent routes or when the `notFound()` function is triggered.

Use the folder hierarchy to define more or less specific error messages depending on the route.

### Example

- Navigate to categories or subcategories that don't exist:
  - [Non-existent category](/not-found/does-not-exist)
  - [Non-existent subcategory](/not-found/electronics/does-not-exist)

Check the official documentation to learn more about custom error pages.
